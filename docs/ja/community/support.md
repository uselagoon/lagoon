# Lagoonコミュニティサポート

LagoonのコミュニティサポートはGitHub Discussionsで提供されています。

Discordでのサポートは終了しました。

_これは現在のサポートチャネルを置き換えるものではありません。既存のサポートチャネルはそのまま維持されます。ここは、他のユーザーやLagoonメンテナーとつながるための場所です。_

バグ、サポート依頼、リポジトリ固有の質問は、該当する[uselagoonリポジトリ](https://github.com/orgs/uselagoon/repositories)にIssueを作成してください。

サポート以外の質問、アイデア共有、一般的なコミュニティ対話は、[GitHub Discussion](https://github.com/uselagoon/lagoon/discussions)をご利用ください。

引き続き、[行動規範](../code-of-conduct.md)と[参加ガイドライン](participation.md)を守ってください。
